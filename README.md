## 📖 О проекте

**Canvas Playground** — это архитектурный фреймворк для быстрого создания и демонстрации анимаций, игр и визуальных эффектов на Canvas 2D. Проект построен на принципах композиции и предлагает гибкую систему для комбинирования готовых компонентов.

### Ключевые особенности

- 🏗 **Компонентная архитектура** — собирайте сложные поведения из простых частей
- 🎯 **Модульность** — каждый пример независим и самодостаточен
- 📦 **Богатая библиотека** — примитивы, компоненты и готовые поведения
- 🎨 **Визуальная демонстрация** — каждый пример сопровождается GIF-анимацией
- 📚 **Документация** — подробное описание каждого примера
- 🚀 **Готов к деплою** — встроенная поддержка GitHub Pages

## 🏗 Архитектура

Проект построен на четырех уровнях абстракции:

### 1. Ядро (Core)
Базовые механизмы работы движка:
- **Engine** — главный управляющий компонент
- **Scene** — управление сценой и объектами
- **GameLoop** — игровой цикл с контролем FPS

### 2. Абстракции (Abstract)
Фундаментальные интерфейсы:
- **GameObject** — базовый класс всех игровых объектов
- **Renderable** — интерфейс отрисовки
- **Updatable** — интерфейс обновления
- **Controllable** — интерфейс управления

### 3. Примитивы (Primitives)
Базовые геометрические фигуры:
- Point, Circle, Rectangle, Line, Triangle, Polygon

### 4. Компоненты (Components)
Поведенческие модули:
- Movement, Rotation, Scaling, Color, Trail, Gravity, Collision

### 5. Поведения (Behaviors)
Готовые комбинации компонентов:
- Bouncing, Orbiting, ParticleSystem, WaveMotion
