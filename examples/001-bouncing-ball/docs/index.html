Компоненты Ball

  • Физика: gravity, bounce, friction
  • Рендеринг: gradient, shadow, highlight
  • След: trail, maxTrail
  • Состояние: isPaused, active

Ключевая структура класса:


class Ball {
    constructor(x, y, config = {}) {
        // Позиция и размер
        this.x = x;
        this.y = y;
        this.radius = config.radius || 20;
        
        // Физические параметры
        this.vx = 0; // скорость по X
        this.vy = 0; // скорость по Y
        this.gravity = config.gravity || 0.6;
        this.bounce = config.bounce || 0.8;
        
        // Визуальные параметры
        this.color = config.color || '#ff6b6b';
        this.trail = [];
    }
    
    update(canvasWidth, canvasHeight) {
        // Обновление физики
        this.vy += this.gravity;
        this.x += this.vx;
        this.y += this.vy;
        
        // Обработка столкновений
        this.handleCollisions(canvasWidth, canvasHeight);
    }
    
    render(ctx) {
        // Отрисовка мяча с эффектами
        this.drawBall(ctx);
        this.drawTrail(ctx);
        this.drawHighlight(ctx);
    }
}

⚙️ Компоненты и их взаимодействие

Каждый компонент отвечает за свою часть функциональности, что упрощает тестирование и модификацию.

🔄

Физический движок

  • Гравитация: постоянное ускорение вниз
  • Упругость: коэффициент отскока от поверхностей
  • Трение: замедление при касании пола
  • Сопротивление: постепенное замедление в воздухе

// Физика мяча
this.vy += this.gravity; // падение
this.x += this.vx;
this.y += this.vy;

// Отскок от пола
if (this.y + this.radius > canvasHeight) {
    this.y = canvasHeight - this.radius;
    this.vy = -this.vy * this.bounce;
    this.vx *= this.friction;
}

Игровой цикл

  • requestAnimationFrame: плавная анимация
  • DeltaTime: независимость от FPS
  • FPS счетчик: мониторинг производительности
  • Столкновения: взаимодействие мячей

function gameLoop(timestamp) {
    const deltaTime = (timestamp - lastTime) / 1000;
    lastTime = timestamp;
    
    // Обновление FPS счетчика
    frameCount++;
    fpsTimer += deltaTime;
    if (fpsTimer >= 1) {
        currentFps = Math.round(frameCount / fpsTimer);
        frameCount = 0;
        fpsTimer = 0;
    }
    
    update(deltaTime);
    render();
    requestAnimationFrame(gameLoop);
}

🎨 Визуализация и эффекты

Все визуальные эффекты создаются программно без использования изображений.

Градиентная заливка

Используется радиальный градиент для создания объемного эффекта. Светлый центр и темные края имитируют освещение.

Блики и тени

Два слоя бликов создают эффект глянцевой поверхности. Тени добавляют глубину и реалистичность.

Система следов

Следы из предыдущих позиций мяча создают эффект движения. Прозрачность затухает со временем.

Визуализация столкновений

При столкновении мячи упруго отталкиваются, создавая реалистичную физику взаимодействия.

Код создания визуальных эффектов:


// Создание градиента
const gradient = ctx.createRadialGradient(
    this.x - this.radius * 0.3,
    this.y - this.radius * 0.3,
    0, // Внутренний радиус
    this.x,
    this.y,
    this.radius // Внешний радиус
);

// Добавление цветовых остановок
gradient.addColorStop(0, '#ffffff'); // Центр - белый
gradient.addColorStop(0.3, this.color); // Основной цвет
gradient.addColorStop(1, 'rgba(0,0,0,0.3)'); // Край - темный

// Применение градиента
ctx.fillStyle = gradient;
ctx.arc(this.x, this.y, this.radius, 0, Math.PI * 2);
ctx.fill();

// Добавление бликов
ctx.beginPath();
ctx.arc(
    this.x - this.radius * 0.25,
    this.y - this.radius * 0.25,
    this.radius * 0.2,
    0, Math.PI * 2
);
ctx.fillStyle = 'rgba(255, 255, 255, 0.4)';
ctx.fill();
