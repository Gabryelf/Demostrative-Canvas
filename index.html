О проекте

Canvas Playground — это коллекция демонстрационных проектов, показывающих возможности HTML5 Canvas 2D API.

Каждый пример создан с использованием чистого JavaScript без сторонних библиотек, картинок или внешних ресурсов — только код и математика.
